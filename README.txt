PROYECTO: Sistema de Gestión de Productos para Vivero

Aprendiz: Laura Combita
Evidencia: GA7-220501096-AA2-EV01

Descripción del proyecto:
Este proyecto consiste en el desarrollo de un módulo de software en Java que permite gestionar productos de un vivero. 
El sistema se conecta a una base de datos MySQL utilizando JDBC y permite realizar operaciones básicas de gestión.

Tecnologías utilizadas:
- Java
- JDBC
- MySQL
- NetBeans
- Maven

Funcionalidades implementadas:
- Conexión a base de datos MySQL mediante JDBC
- Inserción de productos
- Consulta de productos
- Actualización de productos
- Eliminación de productos

Estructura del proyecto:
- conexion: clase para conexión a la base de datos
- modelo: clases del modelo de datos
- dao: acceso a datos (CRUD)
- principal: clase principal del programa

Base de datos utilizada:
Nombre de la base de datos: vivero
Tabla: productos
